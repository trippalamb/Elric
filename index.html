About

This is just a simple molecule viewer. From the perspective of a molecule viewer it really isn't anything special, but what it represents is exciting. It represents my first foray into 3d graphics (utilizing vanilla WebGL). It also happens to be the start of my journey into computational chemistry. By effectively building a game engine to run the visualization of my future computational chemistry projects, I hope to gain a better understanding of what I'm working on and create an opportunity to share my projects as an added bonus.

Eventually this will be a computational chemistry simulation running in the browser. In order to attempt to keep performance at an acceptable level the simlutation backend will be written in C++ compiled to Web Assembley. This gives the added bonus of being reusable for native desktop and server use as well.

In short, this project represents a home base for a multitude of future projects that greatly interest me. Expect to see this page evolve quite a bit over time.
